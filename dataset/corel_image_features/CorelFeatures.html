Corel Image Features

Abstract

This dataset contains image features extracted from a Corel image collection. Four sets of features are available based on the color histogram, color histogram layout, color moments, and co-occurence texture.

Information files:

  • description of the data

Data files:

  • ColorHistogram.asc.gz (4.9M; 20.0M uncompressed)
  • LayoutHistogram.asc.gz (5.0M; 20.0M uncompressed)
  • ColorMoments.asc.gz (2.6M; 6.2M uncompressed)
  • CoocTexture.asc.gz (4.6M; 10.8M uncompressed)
